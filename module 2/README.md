This project is a small demo showing how both symmetric and asymmetric encryption work in Python. The symmetric part uses Fernet, which is basically one shared secret key that can both encrypt and decrypt a message. It’s fast, simple, and good for general data protection. The asymmetric part uses RSA, where a public key encrypts the message and a private key decrypts it. Since RSA only works with bytes, the message gets encoded first, and the padding (OAEP with MGF1 and SHA256) has to match on both encryption and decryption or the message can’t be unwrapped. The program prints out the encrypted data for both methods and then shows the decrypted results so you can see how each system works. Overall, it’s just a quick example of the difference between “one key does everything” (symmetric) and “one key locks, the other key unlocks” (asymmetric).
